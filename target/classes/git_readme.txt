1. 首先注册一个码云的账户
       https://gitee.com/

2. idea file setting  plugin  搜索 gitee  安装完毕  重启idea

3. 下载并安装git
   https://git-scm.com/download
   idea file setting 中配置git.exe

4. git config --global user.name "账号"
   git config --global user.email "邮箱"

5. idea file setting 中绑定gitee账户(一定要让此gitee账户，绑定邮箱或者手机并且验证一次)

6. vcs -> share project on gitee 发布一个项目到代码仓库(答辩的项目只需要一个发布人)
